# Миссия и ценность

- **Кому помогаем:** внутреннему агенту/разработчикам, работающим с IoT проектами на ESP32/ESP8266 с библиотекой Settings by AlexGyver.
- **Почему сейчас:** требуется устойчивый, воспроизводимый Agent Skill для редких кастомных зависимостей Settings и её стека, чтобы ускорить разработку, рефакторинг и отладку.
- **Бизнес-метрики успеха:** сокращение времени на внедрение/отладку Settings, снижение ошибок из-за неполного понимания зависимостей, скорость ответа агента на типовые задачи.

## Scope
- **In-scope:** подготовка и поддержка артефактов для `$skill-creator` (cheatsheet, recipes, templates, playbooks), скрипт обновления с GitHub, интеграция с PlatformIO/Arduino стеком Settings.
- **Out-of-scope:** реализация пользовательских фич вне темы Settings, сторонние UI/HTTP фреймворки, управление CI/CD железом.

## Принципы разработки
- KISS, минимально достаточные изменения; офлайн-готовность, но возможность управляемых обновлений.

## Критерии успеха (верхнеуровневые NFR/SLO/SLA)
- Артефакты обновляются скриптом, покрывая текущие версии Settings и зависимостей.
- Агент использует структурированный контекст (dist/cheatsheet/recipes/templates) и отвечает без ручного поиска в сети.
