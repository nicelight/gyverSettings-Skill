# Ключевые сценарии
- Новый проект: создать PlatformIO/Arduino проект с SettingsGyver + GyverDB + WiFiConnector, собрать UI (виджеты), настроить LittleFS и авто-сохранение.
- Рефакторинг: обновление версий Settings/зависимостей, миграция виджетов/опций, проверка дефайнов (SETT_NO_DB, SETS_NO_CORS) и адаптеров (Gyver/ESP/Async/WS).
- Отладка: диагностика WiFi/AP, файлового менеджера, OTA, websocket/async режимов, проблем RAM/flash и бинарного протокола.
